Занимаюсь версткой Landing-Page

Немного обо мне

Я начинающий веб-разработчик. В основном верстаю Landing-Page. Знания и навыки: + HTML + CSS (Sass) ( Благодаря препроцессору Sass ускоряется верстка сайта) +Валидная верстка (После того как сверстаю блок - сразу же проверяю на валидность через https://validator.w3.org/ это позволяет избегать ошибок валидации, что благоприятно сказывается на загрузку сайта и продвижение сайта) + БЭМ - помогает масштабировать и повторно использовать код, что в свою очередь улучшает производительность. Так же после верстки каждой секции прогоняю через БЭМ дерево https://yoksel.github.io/html-tree/. + Адаптивная верстка - с вашим продуктом, товаром, услугой можно будет ознакомиться на совершенно разных устройствах с разным разрешением экрана, будь то пк, ноутбук, планшет, телефон. + Кроссбраузерность – проверяю верстку через разные браузеры (кроме Internet Explorer) + Основы SEO main/header/nav/footer/section/aside/article + JS / JQuery Разные динамические элементы такие как: Меню бургер Слайдер Модальное окно Табы Аккордеон Валидация формы Parallax Scroll Bar Плавная прокрутка к якорю. Анимации. + Gulp (выполняется рутинные задачи – ускоряет разработку) + Git (помогает вернутся к прошлым решениям, если что-то новое не подошло) +Pixel perfect – верстка будем максимально соответствовать макету (может быть разница в 5 – 10px)
